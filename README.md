Image Resizer

A high-precision image optimization engine for professional workflows.

Image Resizer is a lightweight, powerful desktop application (built with Electron and React) designed to solve the common problems of professional image deployment: stretched logos, pixelated backgrounds, and inconsistent file sizes.

Key Features

Triple-Mode Crop Engine**:
    *   Fit (Letterbox): Preserve 100% of your source pixels. Adds clean black bars to match target ratios without distortion.
    *   Auto (Center-Fill): Mathematically identifies the subject center and fills the frame perfectly.
    *   Manual (Workbench): An interactive precision editor for custom framing.
*   Resolution Presets: Instant hardware matching for Crestron, Extron, Cisco, and Zoom Room hardware.
*   100% Inspect Mode: Verify sharpness and transparency on a pixel-perfect level.
*   Batch Processor: Optimize dozens of assets simultaneously with one click.
*   Safety Overlays: Real-time safe-zone guides to see where UI buttons will cover your image.

Getting Started

Download the App
If you just want to use the tool, download the latest version from the [Releases](https://github.com/ify-gif/Resizer/releases) page.

Developer Setup
If you want to modify the code or contribute:

1.  Clone the Repo:
    ```bash
    git clone https://github.com/ify-gif/Resizer.git
    cd resizer
    ```
2.  Install Dependencies:
    ```bash
    npm install
    ```
3.  Run Development Mode:
    ```bash
    npm run dev
    ```
